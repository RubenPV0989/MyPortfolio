In this section, you will find Python coding examples, most of them focused on automation of IT task.

Name Description Notebook Kaggle

GDAP Advanced Capstone: Predicting Employee Turnover With Logistic Regression This exercise shows a basic use of logistic regression to predict employee turnover based on existing data provided by an HR department
